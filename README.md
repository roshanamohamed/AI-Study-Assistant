# AI Study Assistant  
A conversational AI assistant built using **FastAPI**, **LangChain**, and **Google Vertex AI (Gemini)**.  
The assistant retrieves relevant notes from local files and generates context-aware answers.


# Features
- Conversational AI using Gemini API  
- Simple local text retrieval (notes-based RAG)  
- FastAPI backend with `/ask` endpoint  
- Easy to extend with more documents or vectors
